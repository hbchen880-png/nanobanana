# RunningHub 图片生成工具（批量提交 + 批量轮询版）

这版主要更新：

- API Key 仍然放在“设置”窗口里，主界面不显示
- 支持**先批量提交，再统一轮询**
- 可设置**批量提交并发数**
- 可设置**轮询间隔**
- 应用图标已直接内嵌在程序里
- 打包时会自动生成并嵌入 exe 图标，不需要你手动改图标文件

## 功能

- 选择图片文件夹
- 批量上传并提交 RunningHub 图片编辑任务
- Prompt 可自定义
- aspectRatio 默认 3:4
- resolution 默认 1k
- 结果自动下载
- 可按目标 MB 压缩结果图
- 日志、任务列表、预览都保留

## 推荐使用方式

- 批量提交并发数：建议 3 到 5
- 轮询间隔：建议 3 到 5 秒
- 如果你的 API Key 容量比较大，可以把并发数适当调高
- 如果接口容易限流，就把并发数调低一点

## 启动

### 直接运行源码版
双击：
- `start_app.bat`

### 打包 exe
双击：
- `build_exe.bat`

打包完成后，成品在：
- `dist\RunningHubImageGenerator.exe`

## 图标说明

- 窗口图标已经写进 Python 程序内部
- exe 图标会在打包前自动从程序内部图标生成并嵌入
- 正常使用时不需要单独替换图标文件

## 依赖

- Python 3.10+
- requests
- Pillow

## 注意

- RunningHub 上传接口单张图片限制 10MB
- 上传得到的链接有效期只有 1 天
- 如果压缩值填 0 或留空，则不额外压缩
