# NanoBanana 图片生成工具（增强版）

本版本已按最新需求加入以下功能：

## 任务列表相关
1. **任务列表列顺序调整**：
   - `任务`
   - `状态`
   - `已运行时间`
   - `备注`
   - `输出文件`
2. 原来的 **Task ID 列已取消**，改为 **备注列**。
   - 任务失败时，返回的失败信息会记录到备注列。
3. **任务名前增加小叉（✖）**：
   - 点击任务名前方的小叉可直接删除该任务。
   - 删除后，只有点击 **“刷新文件夹”** 才会重新从图片文件夹载入该图片任务。
4. 设置中新增：**任务列表删除表示删除图片**（默认关闭）
   - 关闭：只从任务列表删除
   - 开启：删除任务时，同时删除对应图片文件（同源任务也会一起移除）
5. 新增 **刷新文件夹** 按钮。

## 生成逻辑相关
1. **源图预览支持拖拽添加**，拖入后可直接点击 **单独生成**。
2. **单独生成输出位置** 可单独设置，默认导出到 **桌面**，支持保存用户设置。
3. **批量输出目录默认规则**：
   - 选择图片文件夹后，默认输出到该图片文件夹**上一级目录**下的 **“香蕉生成”** 文件夹。
   - 若该文件夹不存在，会自动创建。
4. **不再额外生成 originals 子目录**，结果图直接输出到目标输出文件夹。
5. 新增 **“是否删除原图片”**：生成成功后可自动删除原图。
6. 新增 **“生成数量”**：默认 1，可控制每次重复提交多少个任务。
7. 设置中新增 **“最大重试次数”**：默认 3。任务失败时会自动重试，并在状态栏显示类似：
   - `失败1次，重新运行中`
8. 任务列表新增 **已运行时间** 列，会实时更新运行时长。

## 结果图放大 / 对比预览
1. **🔍 放大**：全屏查看结果图。
2. **⚖ 对比**：同时放大查看源图与结果图（左源图、右结果图）。
3. 在放大/对比界面支持：
   - **Q**：退出预览
   - **Esc**：退出预览
   - **R**：重新生成当前源图
   - **Ctrl + 鼠标滚轮**：放大 / 缩小
   - **鼠标拖动图片**：移动位置
4. 当勾选“删除原图片”或原图不存在时，对比按钮会自动隐藏。
